# Agents

This file documents the available Claude Code sub-agents for this project.

## Available Agents

| Agent                 | File                                                          | Purpose                      |
| --------------------- | ------------------------------------------------------------- | ---------------------------- |
| healthsync-guidelines | .claude/agents/healthsync-guidelines/healthsync-guidelines.md | Behavioral coding guidelines |

## Usage

Each agent file contains:

- a `name`
- a `description`
- behavioral instructions
- execution constraints

Agents are intended to complement the main system prompt,
not replace it.
